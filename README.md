# Fency programming language C runtime library
See [Fency Compiler repo](https://github.com/The-Fency-Project/fencyc)
## Build
Just use make.     
`make debug` - debug build (use if you're contributor)       
`make` - release (optimised) build     
`make install` - builds release and installs into `$HOME/.fency/runtime`.    
Recommended if you're building toolchain for usage.
